# Halovecs - Superb Warfare Addon

A sample addon for Superb Warfare mod (version 0.8.8 for Minecraft 1.21.1).

## Description

This is an example addon mod for Superb Warfare, demonstrating how to create custom content that extends the Superb Warfare mod.

## Features

- Basic mod structure for NeoForge 1.21.1
- Dependency on Superb Warfare 0.8.8
- Example mod class structure
- Ready-to-use build configuration

## Requirements

- Minecraft 1.21.1
- NeoForge 21.1.100+
- Superb Warfare 0.8.8
- GeckoLib 4.7.5+

## Building

1. Clone this repository
2. Ensure SuperbWarfare-0.8.8-1.21 project is in the parent directory
3. Run `./gradlew build` (Linux/Mac) or `gradlew.bat build` (Windows)
